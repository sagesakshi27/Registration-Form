# 🌸 Registration Form

A modern and aesthetic **Registration Form** built using **HTML, CSS, and JavaScript**, featuring smooth gradients, input validation, and cute animated popups for success or error messages. 💖  

---

## ✨ Features

- 🎨 **Beautiful Glassmorphism UI** with gradient background  
- 💌 **Email validation** using JavaScript regex  
- 📞 **Phone number validation** (10-digit only)  
- 💬 **SweetAlert2 pop-up** on successful submission  
- 💅 **Responsive design** — works smoothly on all screen sizes  
- 💖 **Animated error messages** with soft pink tones  

---

## 🧠 Technologies Used

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Form structure |
| **CSS3** | Styling & responsive design |
| **JavaScript (Vanilla)** | Validation logic |
| **SweetAlert2** | Pop-up alerts for feedback |

## 💡 Validation Details

| Field | Validation |
|--------|-------------|
| **Email** | Must follow pattern `example@domain.com` |
| **Phone** | Only 10 numeric digits allowed |
| **All other fields** | Required before submission |
